# Recomendações de Compliance

## O que funciona
- Tom: notificação de sistema, NÃO marketing
- Ancoragem em evento transacional real (pedido, sacola, rastreio, conta, pontos)
- Saudações variadas, sempre seguidas de fato transacional
- Idioma 100% do GEO (EN puro para US, ES puro para ES)
- {{first_name}} é a única variável permitida
- Máx 1 emoji neutro/funcional (✅ ℹ️ 🛍️ ok)
- Sem emojis comemorativos (🔥🎉💰🎁 proibidos)

## O que evitar
- Linguagem de marketing direto
- Claims sobre atividade na conta não verificada
- Urgência artificial via status updates falsos
- Mimic de notificações de sistema pra gerar cliques
- Palavras da lista negra: oferta, desconto, grátis (como isca), exclusivo, limitado, últimas horas, imperdível, aproveite, garanta, resgate agora, não perca, sentimos sua falta, volte, novidades, "shop now", "comprar"

## Tornar copy mais forte dentro do compliance
- Ancore num evento transacional real (pontos expirando, ciclo fechando, tier resetando)
- Use urgência implícita via o evento, não via persuasão
- "delivery moves fast" ou "check before the cycle resets" — implicações factuais
- Se não existe evento real, recomende criar um (pontos, fidelidade, cadastro) em vez de fabricar urgência
