## Workflow — Novo Template

Ao criar um novo template, siga obrigatoriamente este fluxo.

### 1. Brainstorm

Antes de gerar qualquer layout:

- Entenda o objetivo do criativo.
- Sugira diferentes abordagens visuais.
- Discuta vantagens e desvantagens de cada ideia.
- Trabalhe junto com o usuário até definir a melhor direção.
- Nunca gere um template sem antes validar o conceito quando a direção ainda não estiver clara.
- Nunca gere coisas como espelho, tela de celular, e coisas que tiram foco do criativo, como price tag tb
 
### 2. Compliance

Todo template deve seguir as regras de Compliance Meta.

Em caso de conflito entre estética e compliance, compliance sempre tem prioridade.

### 3. Estrutura

Todo template deve conter obrigatoriamente:

- Título (Headline)
- Subtítulo (Subtitle)
- CTA

A hierarquia visual deve deixar essa estrutura clara.

### 4. Copy

- Utilizar apenas copies previamente aprovadas.
- Nunca criar uma nova copy sem solicitação.
- Caso nenhuma copy existente seja adequada, apresentar opções para discussão antes de escrever uma nova.

### 5. Design

O design deve priorizar legibilidade.

Regras obrigatórias:

- Fundo sempre claro ou com contraste suficiente para a copy.
- A copy deve ser facilmente legível.
- O CTA deve possuir destaque visual.
- A composição deve manter equilíbrio entre texto e elementos gráficos.

### 6. Elementos visuais

Os elementos utilizados devem remeter ao universo da moda e compras.

Exemplos recomendados:

- mulher utilizando roupas
- roupas
- cabides
- sacolas de compras
- gift box
- popup de alerta
- etiquetas
- caixas de entrega
- vitrines
- acessórios
- telas de interface relacionadas à compra

### 7. Elementos proibidos

Nunca utilizar elementos que não façam parte do contexto natural da oferta.

Exemplos:

- cartas
- envelopes
- bilhetes
- calendários
- convites
- documentos
- certificados
- qualquer objeto que tente representar uma comunicação oficial da SHEIN

### 8. Consistência

Cada novo template deve possuir identidade própria.

Variar:

- composição
- cores
- disposição dos elementos
- estilo gráfico

Sem perder:

- identidade visual
- legibilidade
- compliance
- padrão aprovado
